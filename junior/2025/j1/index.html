Problem Guide

A solution to the first problem requires a conditional statement such as an if statement. When the condition 
N<=C*P is true, yes should be printed, and otherwise no should be printed.

Solution Code


#include<iostream>
using namespace std;
int main(){
    int n, c, p;
    cin>>n>>c>>p;
    if(n<=(c*p)){
        cout<<"yes";
    }else{
        cout<<"no";
    }
    return 0;
}
