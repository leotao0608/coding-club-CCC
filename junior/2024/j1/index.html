Problem Guide

The first problem on this year's contest involves printing out the result of an arithmetic expression. The expression 
consists of three input values representing the number of plates of each colour, and constant values which are the cost 
of each colour of plate.

Solution Code


#include <bits/stdc++.h>
using namespace std;
int main() {
    int a, b, c;
    cin >> a >> b >> c;
    cout << a * 3 + b * 4 + c * 5;
    return 0;
}
